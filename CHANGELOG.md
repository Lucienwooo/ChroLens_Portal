# ChroLens_Portal 版本更新記錄

## v2.4 (2025/11/19)

### ✨ 新增功能
- **自動更新系統**
  - 整合 GitHub Releases 自動更新機制
  - 支援一鍵檢查更新
  - 自動下載和安裝更新包
  - 即時顯示下載和安裝進度
  - 自動備份舊版本資訊

### 🎨 UI 優化
- 關於視窗新增「檢查更新」按鈕
- 更新對話框顯示版本資訊和更新內容
- 進度條即時顯示更新狀態

### 🔧 技術改進
- 新增 `update_manager.py` - 更新管理核心
- 新增 `update_dialog.py` - 更新對話框 UI
- 版本號統一管理 (`CURRENT_VERSION`)

### 📦 開發工具
- 新增 `build_simple.py` - Python 打包腳本
- 新增 `打包.bat` - 一鍵打包批次檔
- 優化專案結構，移除冗餘文件

## v2.3 (2025/11/18)

### ✨ 新增功能
- **排程功能**
  - 支援設定時間自動啟動指定分組
  - 批量刪除排程任務
  - 整合 Windows 工作排程器
  - 時間選擇改為下拉式選單

- **視窗佈局記憶（FancyZones）**
  - 一鍵捕獲所有分組視窗的位置、大小、狀態
  - 自動恢復視窗佈局
  - 支援多螢幕配置
  - 記住視窗的正常/最大化/最小化狀態

- **管理員權限**
  - 自動請求提升為管理員權限
  - 確保所有視窗操作正常運作

### 🎨 UI 優化
- 縮減視窗高度（760 → 720）
- Mini 模式優化
- 刷新視窗按鈕改為 emoji 圖示 🔄
- 排程對話框 UI 改進

### 🔧 Bug 修復
- 修正快捷鍵無法觸發的問題
- 修正視窗置頂後縮小的問題
- 快捷鍵從 `Ctrl+1~6` 改為 `Alt+1/2/3/Q/W/E`
- 使用 `SWP_NOZORDER` 和 `SWP_NOOWNERZORDER` 防止視窗大小改變
