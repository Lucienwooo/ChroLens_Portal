## 🔄 自動更新系統

### 使用方式

1. **檢查更新**
   - 點擊「關於」→「檢查更新」
   - 系統自動連接 GitHub Releases 檢查最新版本

2. **自動下載安裝**
   - 發現新版本時顯示更新內容
   - 點擊「立即更新」開始下載
   - 下載完成後自動安裝並重啟程式

3. **手動更新**
   - 前往 [GitHub Releases](https://github.com/Lucienwooo/ChroLens_Portal/releases)
   - 下載最新版本 ZIP 檔案
   - 解壓縮並執行

## 📝 注意事項

1. **系統需求**
   - Windows 10/11
   - Python 3.8+ (如使用原始碼)
   - 建議使用打包後的 .exe 檔案

2. **更新建議**
   - 更新前建議備份 `chrolens_portal.json` 設定檔
   - 更新後首次執行可能需要重新設定快捷鍵

3. **已知問題**
   - 部分防毒軟體可能誤報，請加入白名單
   - 某些系統視窗（如 UAC）無法操作屬正常現象
